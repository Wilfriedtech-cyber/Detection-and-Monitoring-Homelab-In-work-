I used a Dell  desktop with 64 GB RAM and 500 SSD GB and I installed vmware esxi onto it. Of course you can use other machines but i would recommend to have a 64 gb ram and at least 500GB for storage as VMWARE ESXI By itself took almost 200 GB.
After I installed vmware ESXI, now was the time to upload the virtual machines and install them.They should be pretty straight forward when it comes to installing a virtual machine on vmware esxi and I basically went with the default configurations for each machines up until I have got to where I needed to assign storage and network group.
